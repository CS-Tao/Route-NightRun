# 夜跑小助手App
夜跑小助手是一款提供多样的夜跑服务的安卓应用软件，可以为您在安全的地方规划指定长度的路线。
## 平台要求
- 工具：Android Studio
- SDK版本：Android SDK 23
- 运行时最低系统版本：Android 6.0
## 使用方法
1. 克隆本项目，使用
    ```
    git clone https://github.com/HPDell/Route-NightRun.git
    ```
    进行克隆，或者下载GitHub提供的zip文件。
2. 在Android Studio中打开，并编译。
3. 完成后即可部署在安卓手机上。
